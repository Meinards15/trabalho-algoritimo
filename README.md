# Como instalar o programa
Utilizando o Gradle e o Git possivel executar o programa atraves do terminal, segue o passo a passo:

`git clone https://github.com/meinards15/trabalho-algoritimo.git`

`cd trabalho-algoritimo`

`./gradlew run --console=plain`

(Certifique-se que possui o Java 21 instalado no seu sistema, Caso não o possua você pode clicar [aqui](https://download.oracle.com/java/21/archive/jdk-21.0.10_windows-x64_bin.exe) para instalar-lo.)
